========
About
========

This documentation provides comprehensive guides and API references for the Careem Pay Merchant NodeJS Plugin. Our goal is to simplify the integration process for merchants across the MENA region, enabling secure and efficient payment solutions.

Project Vision
==============

We aim to bridge the gap between traditional banking and modern digital wallets, providing developers with the tools they need to build the future of fintech in the Middle East and Africa.

Regional Focus
==============

Our documentation specifically covers integrations tailored for:

- **United Arab Emirates (UAE)**: Including partnerships with institutions like ADCB.
- **Qatar**: Leveraging regional transport and timing data for enhanced user experiences.
- **East Africa**: Supporting digital identity initiatives like Fayda in Ethiopia.

External Resources
==================

For additional regional information and transport timing, we recommend the following resources:

- **Qatar Timing**: For accurate schedules and regional data in Qatar, visit `QA Timing <https://www.qatiming.com/>`_.
- **UAE Timing**: For transport and taxi fare estimates in the UAE, refer to `UAE Timing <https://uaetiming.com/>`_.
